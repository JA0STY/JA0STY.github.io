This is a website

To make a website, first you need to download visual studios, then you can go to your command prompt which is built into your computer, then you can either make a directory with the command mkdir or you can go to your desktop with the command cd desktop. After you have made it to your directory, you now need to make a new folder and you can name it what you want, after making a new directory in your desktop or anywhere on your computer, you then need to use the command cd and what you named the directory, for example, cd website. Now there is one more step before you can start customizing your website, which is to make it. Use the command echo.>index.html by using this code, the website will be created. You can go to your files and look at it although there will be nothing on it since you have done nothing to it yet.

So you have made your website but now you want to customize it, this is where it gets a little harder. First you need to open visual studios and select view, open file, and then the website that you named, for example, view-open file-code. The first thing you need to do is to type doctype.html and it should show up as a blue text in all caps.Then you will press enter and type html.I will explain how to change colors and add text to your website. First you will type head and it will show another behind your cursor, then you can press enter 3 times and go to the first enter. Then, type style and it will show up as 2 again press enter and go up once and type background-color and it will let you pick a color. Next go under the second head name and type body. Now press enter 5-10 times and go to the first space. Type h1 which means heading 1 and you can type whatever you want and it will show up on your web page. You can continue doing this 9 times if you want, after that, you can go under the heading or headings and type p this means paragraph and again you can type what you want but it doesnt get smaller as it goes on unlike the headings. Lastly press ctrl and s at the same time to save what you have created, and go to files on your computer, and search the name of your webpage and open it if you haven't already, it will show you everything you have done, and if you opened it before, simply refresh the page and you will see everything. The last thing I will tell you is how to add images, first go back to visual studios, and go to the body tag. Type img src= and it will show you quotation marks. Go to your browser and find any image. Once you find an image, right-click on it and look for copy image address and pick that. Then go back to visual studios, and paste it in between the quotation marks and press ctrl and s at the same time again. Now go to your web page, and the image you selected is now there.
